# BenReg's Reapack
URL to import in Reaper : https://github.com/BenReg/Reascripts/raw/master/index.xml \
Some Reascripts for Reaper and communication with Wwise via WAAPI\
WAAPI needs to be activated in the Wwise user preferences and ReaWwise needs to be installed

* WwiseToReaper : will import the files selected in the Wwise MediaPool on the current selected track in Reaper
